Global Economical Data Analysis

A Basic Data Analysis report, which describes various factors such as GDP, Inflation rate, Unemployment, Imports and Exports of goods and services along with trade balances.
This report contains data from various countries (China, Germany, India, Japan, Russia, and USA) From the Financial years of 2012 to 2022.

This data set is taken from World bank open data source.
